val game_init : int -> t

game_init n is a board of size n x n with all Men pieces set up in their correct starting positions. Requires: n is between 8 and 26 inclusive

val get_board : t -> square list list
val terminal_rep_string : t -> int -> string

terminal_rep_string board count is a string representation of the board board with row and column labels starting at count. For example, terminal_rep_string board 1 will have labels for the rows from 1 to size of the board and column labels starting at 'A'. Requires: count > 0

val count_inactive : t -> color -> int

count_inactive p_color is the total number of pieces which have color p_color and are not on the board
